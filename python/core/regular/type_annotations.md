# Введение
Type annotations (аннотации типов) — это способ указания типов переменных, аргументов функций и возвращаемых значений в языке Python. Аннотации помогают улучшить читаемость кода, облегчить отладку и работу с инструментами статической проверки типов.

# Основные понятия

## Зачем нужны type annotations?
Аннотации типов не изменяют работу кода, но они предоставляют полезную информацию разработчикам и инструментам анализа кода. Основные преимущества использования аннотаций типов:
- Улучшение читаемости кода.
- Облегчение отладки и рефакторинга.
- Возможность использования инструментов статической проверки типов, таких как mypy.
- Повышение надежности и предсказуемости кода.

## Базовый синтаксис
Базовый синтаксис аннотаций типов в Python достаточно прост. Аннотации указываются после двоеточия `:` и перед знаком равенства `=` для переменных, и после двоеточия для аргументов функций и возвращаемых значений.

Пример:
```python
def greeting(name: str) -> str:
    return 'Hello ' + name

age: int = 25
```

# Примеры использования

## Аннотации для переменных
Аннотации типов для переменных указываются после имени переменной и двоеточия.

Пример:
```python
name: str = "Alice"
age: int = 30
height: float = 1.75
```

## Аннотации для функций
Аннотации типов для функций указываются для каждого аргумента и для возвращаемого значения.

Пример:
```python
def add(a: int, b: int) -> int:
    return a + b

def concatenate(strings: list[str]) -> str:
    return ''.join(strings)
```

## Аннотации для классов
Аннотации типов можно использовать и в классах, указывая типы атрибутов класса.

Пример:
```python
class Person:
    name: str
    age: int

    def __init__(self, name: str, age: int) -> None:
        self.name = name
        self.age = age
```

# Стандартные типы и их использование

## Простые типы
В Python существуют базовые типы данных, которые часто используются в аннотациях:
- `int` для целых чисел
- `float` для чисел с плавающей точкой
- `str` для строк
- `bool` для булевых значений

Пример:
```python
x: int = 10
pi: float = 3.14
message: str = "Hello"
is_valid: bool = True
```

## Коллекции
Python поддерживает несколько встроенных типов коллекций, которые также можно использовать в аннотациях:
- `list` для списков
- `tuple` для кортежей
- `dict` для словарей
- `set` для множеств

Пример:
```python
numbers: list[int] = [1, 2, 3]
coordinates: tuple[float, float] = (1.0, 2.0)
user_info: dict[str, str] = {"name": "Alice", "city": "Wonderland"}
unique_numbers: set[int] = {1, 2, 3}
```

## Объединение типов и необязательные типы
Иногда нужно указать, что переменная может иметь несколько типов или быть необязательной. Для этого используются `Union` и `Optional`.

Пример:
```python
from typing import Union, Optional

number: Union[int, float] = 4.5
optional_number: Optional[int] = None
```

# Типы из модуля `typing`

## Типы коллекций
Модуль `typing` предоставляет более мощные возможности для типизации коллекций.

Пример:
```python
from typing import List, Tuple, Dict, Set

numbers: List[int] = [1, 2, 3]
coordinates: Tuple[float, float] = (1.0, 2.0)
user_info: Dict[str, str] = {"name": "Alice", "city": "Wonderland"}
unique_numbers: Set[int] = {1, 2, 3}
```

## Union и Optional
Модуль `typing` предоставляет `Union` для объединения типов и `Optional` для указания необязательных типов.

Пример:
```python
from typing import Union, Optional

number: Union[int, float] = 4.5
optional_number: Optional[int] = None
```

## Типизация функций
Модуль `typing` также предоставляет инструменты для более точной типизации функций, такие как `Callable`.

Пример:
```python
from typing import Callable

def greet(name: str) -> str:
    return f"Hello, {name}"

def process(func: Callable[[str], str], name: str) -> str:
    return func(name)

print(process(greet, "Alice"))
```

# Проверка типов и инструменты

## Mypy
Mypy — это инструмент статической проверки типов для Python. Он проверяет соответствие типов аннотаций и реальных значений.

Пример использования:
```bash
mypy script.py
```

## Pyright
Pyright — это еще один инструмент для проверки типов, разработанный Microsoft. Он также предоставляет интеграцию с VSCode.

Пример использования:
```bash
pyright script.py
```

## Интеграция с IDE
Современные IDE, такие как PyCharm и VSCode, поддерживают аннотации типов и могут использовать инструменты, такие как mypy и Pyright, для проверки типов в реальном времени.

# Заключение
Type annotations в Python — это мощный инструмент, который помогает улучшить читаемость, надежность и поддержку кода. Они предоставляют полезную информацию разработчикам и инструментам анализа кода, облегчая отладку и предотвращение ошибок. Использование аннотаций типов рекомендуется во всех проектах для повышения их качества.
